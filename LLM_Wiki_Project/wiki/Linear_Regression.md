# Linear Regression

Linear regression is a technique that models the linear relationship between a continuous predictor variable (X) and a continuous outcome variable (Y). 

## Core Components
- **Intercept**: Expected value of Y when X is 0.
- **Slope**: The rate of change in Y for a one-unit change in X.
- **Residuals (Error)**: The difference between observed values and values predicted by the model.

## Evaluation
The model's goodness of fit is evaluated using an F-test, while the significance of individual predictors is tested with t-tests. The effect size, R-squared, indicates the proportion of variance in the outcome explained by the predictor(s).
