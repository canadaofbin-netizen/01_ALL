# Form-Factor Fragmentation

Form-Factor Fragmentation occurs when an interaction ecosystem forces the user to wear or utilize mandatory secondary devices (auxiliary peripherals), thereby breaking the seamless, self-contained experience of ambient computing.

## The Smart Glasses Dilemma
As detailed in the [[BCI_Roadmap]], a prime example of this fragmentation is the industry's approach to circumventing the [[Volume_Conduction]] limits of temporal EEG. For instance, Meta proposed using high-density electromyography (HD-EMG) wristbands to decode precise finger movements. While this solves the directional input problem, it forces the user to wear an extra device on their wrist. 

True spatial computing requires an **'All-in-One'** form factor (Hands-free / Wrist-free), where interactions are entirely self-contained within the smart glasses themselves, often achieved through innovations like [[Asymmetric_Hybrid_Control]].
