# Non-Parametric Tests

Non-parametric tests are "distribution-free" alternatives to standard parametric tests (like [[ANOVA]] or [[T_Test]]). They are highly useful when data violate normality, contain severe outliers, or when analyzing small sample sizes.

## Common Methods
Instead of raw scores, these tests often rank data points:
- **Wilcoxon Rank-Sum / Mann-Whitney U**: For independent samples.
- **Wilcoxon Signed-Rank**: For paired samples.
- **Kruskal-Wallis**: For comparing three or more groups.
