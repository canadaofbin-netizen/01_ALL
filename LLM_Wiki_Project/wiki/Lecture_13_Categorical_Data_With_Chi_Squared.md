# Lecture 13: Categorical Data with Chi Squared

## Overview
Categorical variables describe characteristics (nominal or ordinal). The Chi-Squared test (χ²) evaluates whether observed frequencies differ from expected frequencies under the null hypothesis.
*(Source: Lecture 13. Categorical data with chi squared - notes.pdf)*

## Goodness of Fit Test
Tests if observed data fits expected frequencies (e.g., equal distributions). 
- Null Hypothesis: Expected probabilities match a specific distribution.

## Test of Independence
Tests for an association between two categorical variables using a contingency table.
- **Expected Frequencies**: Calculated as `(Row Total * Column Total) / Grand Total`.
- **Effect Sizes**: Phi (for 2x2 tables) and Cramer's V (for larger tables).

## Assumptions
- Data are frequencies/counts.
- Categories are mutually exclusive.
- Independent study groups.
- Expected cell frequencies should ideally be > 5.
