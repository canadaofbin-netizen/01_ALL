# Motor Imagery (MI)

Motor Imagery (MI) involves the mental execution of a movement without any overt muscle activity, generating Event-Related Desynchronization (ERD) signals in the mu and beta rhythms of the motor cortex.

## Hybrid Paradigms
A major challenge in [[Brain_Computer_Interface]] applications is "BCI illiteracy"—where users fail to generate decodable ERD signals. To combat this, recent research has shifted to hybrid paradigms. Combining Motor Imagery with Overt Spatial Attention (OSA) allows users to modulate multiple cortical networks simultaneously. This drastically improves accuracy for tasks like continuous 2D cursor navigation in virtual environments.
