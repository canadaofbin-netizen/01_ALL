Lecture 11. Linear Regression 
 
In Lecture 9, we learned that a correlation tests for an association between two 
continuous variables. We also learned that finding a 
correlation does not mean that we have proven that one 
variable affects the other; in other words, correlation does not 
imply causation.  
 
A linear regression is a more complex type of analysis that also works with two 
continuous variables, but with a crucial advantage : it involves building a statistical 
model that describes how  changes in one variable (X) 
correspond to changes in the other variable (Y). This makes it 
a highly useful tool, as it effectively enables you to make 
predictions about the value of Y (the outcome) based on a 
given value of X (the predictor).  
Note: The linear regression assumes that the relationship between X and Y is linear.  
Other types of relationships (e.g., cubic, quadratic…) will require a different type of 
model. For now, we’ll focus only on linear relationships.

How to build a linear model? 
The linear model (or regression line, or line of best fit) can be expressed 
mathematically using three numbers: the intercept, the slope, and the error.  
Intercept 
The intercept is the value of Y when X=0. 
In other words, it’s the starting point of 
the regression line.  
In our example above, the intercept is 52, 
meaning that a student who makes 0 
homework submissions is expected to 
get around 52% on their final exam.  
Slope 
The slope expresses how much the value 
of Y changes with each degree of X. Each 
time X increases by 1 point, Y increases 
by the value of the slope.  
In our example, the slope is 2, meaning 
that with each new homework 
submission your exam score increases 
by 2%.  
We can use the intercept and the slope to predict what the outcome variable will be 
based on any value of the predictor we choose. To find the value of the outcome 
variable, we just need to start with the intercept (the starting point of the regression 
line) and then add the slope as many times as the value of the predictor.  
Question: In our example with the homework submission s and the exam scores, we 
know the intercept is 52 and a slope is 2. What exam score will a student get if she 
makes 3 homework submissions?

Error 
Our regression line is a statistical model – a simplified description of the structure of 
the data. Because it is simplified, a model never describes the data perfectly, so the 
actually observed datapoints will deviate somewhat from the values predicted by the 
regression line. This discrepancy between the linear model and the real data is called 
the error.

Testing hypotheses with regression 
There are two types of hypothesis test for regression models:  
1. Does the model fit the data? To test this hypothesis, we want to see whether the 
variance explained by the model is greater than the residual variance.  
2. Is there a significant association between a predictor and the outcome? We test 
this using a t-test to see if the slope is significantly different from 0. 
In the case of simple linear regression , these are two ways of asking the same thing 
and we will get the same result for both.

We can express the degree to which out model fits the data using an F-test, which 
measures the ratio of two variances:  
Mean Square s of the  Model = the variance explained by the 
model (deviation of predicted values from the mean value)  
Mean Square s of the  Error = the variance not explained 
(deviation of observed values from predicted values)  
 
• H0: The model is no better than the mean for predicting Y 
• H1: The model is better than the mean for predicting Y 
The F value has its own null hypothesis distribution which is used to determine the 
p-value. A significant F test (p<.05) means the null hypothesis is rejected.   
 
Effect size: R2 
The R2 value is the effect size measure which tells you the proportion of the variance 
in Y that is accounted for by the model . For example, R2=.26 means that 26% of the 
variation in exam scores is explained by the number of homework submissions – 
leaving another 74% of the variation in exam scores unaccounted for by the model. 
Obviously, the bigger the R2, the better X is at predicting Y.

t-tests for the slope 
T-tests can be used to test whether each coefficient (intercept and slope) is different 
from some expected value (usually 0). The intercept being different from 0 is not very 
informative, but the slope is. It allows us to test the null hypothesis that there is no 
relationship between the predictor and outcome  – a significant t -test for the slope 
means the null hypothesis that the slope is 0 (no relationship between predictor and 
outcome) is rejected.

Assumptions of linear regression 
As with correlation:  
• Interval or ratio level data 
• Data points available for each participant for each of the two variables 
• Linear relationship between the two variables 
• Homogeneity of variance: the residuals have same distribution (and standard 
deviation of residuals is same for all values of X and Y) 
• No extreme outliers – can distort the relationship between two variables 
Plus: Residuals must be normally distributed – NOTE: you will learn how to do this in
